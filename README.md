## ‚ú® Features

- **Interactive Messaging**: The "No" button cycles through 10 cute, pleading messages to convince them otherwise.
- **Growing "Yes" Button**: With every refusal, the "Yes" button grows larger and larger, making it impossible to miss!
- **Celebration Page**: A dedicated success page with a celebratory GIF when they finally say "Yes".
- **Responsive Design**: Looks great on desktop and mobile screens.

## üöÄ How to Use

1. **Download the code**: Clone the repository or download the ZIP file.
2. **Open the project**: Simply double-click on `index.html` to open it in your web browser.
3. **Customize** (Optional):
   - **Messages**: Edit the `messages` array in `script.js` to add your own inside jokes or sweet nothings.
   - **Images**: Change the `src` attribute of the `<img>` tags in `index.html` and `yes_page.html` to use different GIFs.
4. **Send it**: host it on GitHub Pages, Vercel, or Netlify to send a link, or just show it to them on your laptop/phone!
